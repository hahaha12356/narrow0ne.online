Narrow One Patch Notes & Update Analysis – December 2025

This monthly recap highlights noteworthy changes and practical takeaways so you can adapt quickly.

Highlights

  • New map cover additions reduce dominant sniper lines on select choke points.
  • Minor bow balance tweaks: slightly more consistent draw timing.
  • Quality-of-life fixes: UI readability and small performance improvements.

Balance Impact

Extra cover creates more mid-fight options for flanks; expect fewer long-range lockouts. The draw consistency helps newer players hit more reliable releases while preserving high-level skill expression.
